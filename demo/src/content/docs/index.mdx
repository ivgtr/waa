## 機能

<CardGrid>
  <Card title="Pitch 保持 time-stretch" icon="rocket">
    WSOLA ベースのストリーミング再生で、リアルタイムにテンポを変更。処理は別スレッドで実行されます。
  </Card>
  <Card title="BYO AudioContext" icon="setting">
    すべての関数が AudioContext を第一引数に取ります。隠れた global state はありません。
  </Card>
  <Card title="Tree-shake 対応" icon="seti:json">
    11 の独立モジュールと個別エントリポイント。必要なものだけインポートできます。
  </Card>
  <Card title="Framework adapter" icon="puzzle">
    React、Vue、Svelte など、あらゆる framework 向けの snapshot ベース統合を内蔵。
  </Card>
</CardGrid>
