## 機能

<CardGrid>
  <Card title="ピッチ保持タイムストレッチ" icon="rocket">
    WSOLA ベースのストリーミング再生で、リアルタイムにテンポを変更。処理は別スレッドで実行されます。
  </Card>
  <Card title="BYO AudioContext" icon="setting">
    すべての関数が AudioContext を第一引数に取ります。隠れたグローバルステートはありません。
  </Card>
  <Card title="ツリーシェイク対応" icon="seti:json">
    11 の独立モジュールと個別エントリポイント。必要なものだけインポートできます。
  </Card>
  <Card title="フレームワークアダプター" icon="puzzle">
    React、Vue、Svelte など、あらゆるフレームワーク向けのスナップショットベース統合を内蔵。
  </Card>
</CardGrid>
